# CI Targets

- Linux: `x86_64-unknown-linux-gnu`
- macOS: `x86_64-apple-darwin`, `aarch64-apple-darwin`
- Windows: `x86_64-pc-windows-msvc`
- FreeBSD: cross-compile target hook
- OpenBSD: cross-compile target hook
- NetBSD: cross-compile target hook
- DragonFly BSD: cross-compile target hook
- illumos-like targets: cross-compile target hook
